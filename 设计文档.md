# 全自动市场热点分析与盘前前瞻系统 — 设计文档

**日期:** 2025-05-29
**版本:** v1.0

## 概述

构建全自动美股市场分析系统，每日 21:00（北京时间）发送市场热点分析与盘前前瞻，每周六 09:00 发送本周总结复盘。分析质量对标高盛级别，最终以中文邮件发送到 547230316@qq.com。

## 架构

```
GitHub Actions (云端调度，无需本地开机)
├── Cron: 0 13 UTC (每日 21:00 BJT) → 日报脚本
└── Cron: 0 1 UTC Sat (周六 09:00 BJT) → 周报脚本
         │
         ▼
    Python 分析引擎 (market_analyzer.py)
    ├── 数据采集层
    │   ├── yfinance: 美股实时/历史行情、板块ETF、期货

        └── QQ邮箱 SMTP (smtp.qq.com:465, SSL)
```

## 技术栈

| 层 | 技术 | 原因 |
|---|---|---|
| 调度 | GitHub Actions | 免费、云端、无需电脑开机 |
| 数据 | yfinance | 免费、美股数据全面 |
| 新闻 | NewsAPI + web-access/anysearch | 英文财经新闻源 |
| AI分析 | DeepSeek v4 Pro (阿里云百炼) | 推理能力强、中文输出好 |

## 日报模板

1. 当日大盘综述（标普/纳指/道指涨跌幅 + 关键点位）
2. 市场热点扫描（涨幅最大板块 TOP5 + 驱动因素）
3. 个股异动追踪（涨跌幅异常 + 成交量异动 + 机构动向）
4. 次日盘前前瞻（期货指向 + 经济数据日历 + 财报预警）
5. 风险提示

## 周报模板

1. 本周市场全景（五大指数周涨跌 + 板块轮动）
2. 核心主题回顾（本周主导叙事 3-5 条）
3. 关键宏观事件（美联储/CPI/就业 + 市场反应）
4. 资金流向分析（板块资金流 + 机构动向）
5. 下周前瞻（经济日历 + 关键财报 + 技术面）
6. 高盛级别策略研判

## 安全设计

- 所有密钥通过 GitHub Secrets 存储
- QQ 授权码、API Key 不写入代码
- 脚本运行时从环境变量读取
